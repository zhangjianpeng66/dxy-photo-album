## 三、怎么改内容

### 3.1 改文字（标题、文案）

1. 打开 https://github.com/zhangjianpeng66/dxy-photo-album
2. 点进 `config.json` → 点右上角 ✏️（铅笔图标）编辑
3. 修改里面的文字，格式如下：

```json
{
  "title": "dxy",                          // ← 页面主标题（左上角）
  "subtitle": "forever lovers  jun zi hao qiu",  // ← 副标题

  "cards": [
    {
      "title_cn": "蓝调时刻",      // ← 中文标题
      "title_en": "Blue Hour",     // ← 英文副标题
      "text": "她和湖，都很安静",   // ← 文案正文
      "image": "photos/1.jpg",     // ← 照片文件名
      "color": "#304060"           // ← 照片区背景色（照片未加载时显示）
    }
    // ... 更多卡片
  ]
}
```

4. 改完点 **"Commit changes"** → 输入描述 → 点 **"Commit changes"**
5. 等约 10 秒，刷新页面即可看到更新

### 3.2 添加照片

1. 把照片重命名为 `17.jpg`、`18.jpg`……（和已有照片编号接续）
2. 打开 GitHub 仓库的 `photos/` 文件夹
3. 点 **"Add file"** → **"Upload files"** → 拖入新照片
4. 点 **"Commit changes"**
5. 在 `config.json` 的 `cards` 数组末尾添加新卡片配置
6. 手动更新服务器（见下方「手动更新服务器」）

### 3.3 替换照片

1. 在 GitHub 仓库的 `photos/` 文件夹，找到要替换的照片
2. 点照片 → 右上角 ⋯ → **"Delete file"** → 提交
3. 上传同名新照片
4. 手动更新服务器

## 五、常见问题

| 问题 | 解决方法 |
|------|---------|
| 页面打不开 | 检查 ECS 是否开机，安全组 80 端口是否开放 |
| 照片不显示 | 检查照片文件名和 config.json 里的 `image` 路径是否一致 |
| 3D 卡顿 | 手机性能有限属于正常；电脑卡顿刷新页面 |
| 想改主题色 | config.json 里每张卡片的 `color` 字段 |
| 页面太暗/太亮 | 目前固定暗色美学，想调整告诉我 |
